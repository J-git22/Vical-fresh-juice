# Vical Freshly Squeezed 🍊

A beautifully designed, premium e-commerce landing page built for **Vical Freshly Squeezed**, a business dedicated to providing 100% real, authentic fruit juices with zero artificial preservatives or added sugars.

## 🌟 Features

- **Premium Modern Design**: Features a clean aesthetic utilizing soft gradients, glassmorphism (frosted glass) effects, and deep shadows for a high-end feel.
- **Fully Responsive Layout**: Perfectly adapts to all screen sizes, from large 4K monitors to small mobile phones.
- **Direct WhatsApp Integration**: Users can order juices, request wholesale packages, or leave reviews directly through pre-filled WhatsApp messages.
- **Interactive Galleries & Modals**: Smooth image pop-ups to showcase products, services, and the production process in high detail.
- **Comprehensive Sections**:
  - **Freshness & Vitality**: Highlights the health benefits of 100% natural juices.
  - **Shop/Menu**: Displays the available juice blends.
  - **Wholesale**: Options for events and large gatherings.
  - **Production Process**: A transparent look at how the juice is made, from handpicking to bottling.
  - **Services/More**: Interactive cards showcasing catering and custom packages.
  - **Testimonials**: Customer reviews building trust.

## 🛠️ Technology Stack

This project was built from scratch without heavy frameworks to ensure blazing fast load times and ultimate control over the aesthetic.

- **HTML5**: Semantic structure and accessibility.
- **CSS3 (Vanilla)**: All styling, including complex grid layouts, animations, and responsive media queries. No external CSS frameworks were used.
- **JavaScript (Vanilla)**: Handles the image modal logic and the dynamic WhatsApp URL generation for the "Leave a Review" and ordering buttons.
- **Phosphor Icons**: Lightweight, beautiful iconography used throughout the site.

## 🚀 How to Run Locally

You do not need a server or build process to run this website!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jerryverse/Vical-fresh-juice.git
   ```
2. **Navigate to the folder:**
   ```bash
   cd Vical-fresh-juice
   ```
3. **Open `index.html`:**
   Simply double-click the `index.html` file to open it in your default web browser.

## 📱 Screenshots & Views
The website utilizes CSS Grid and Flexbox to ensure that complex layouts—like the split "Production Process" and "Wholesale" sections—gracefully stack into single columns on mobile devices, providing a seamless browsing experience.
